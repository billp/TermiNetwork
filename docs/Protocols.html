Protocols

The following protocols are available globally.

  • TNErrorHandlerProtocol

    Use this protocol to create error handlers that can be passed to TNConfiguration instances. Every class that implements this protocol will be used to handle errors when a request is failed.

    This protocol is used to register a middleware in order to modify body and headers of a request. (e.g. it can be used with Crypto Swift library to encrypt or decrypt the data.
