Initializers

  • init(method:url:headers:params:configuration:)

    Initializes a TNRequest request.

    parameters:

    • method: The http method of request, e.g. .get, .post, .head, etc.
    • url: The URL of the request.
    • headers: A Dictionary of header values, etc. “Content-type”: “text/html”
    • params: A Dictionary as request params. If method is .get, it automatically appends them to url, otherwise it is seting them as request body.
    • configuration: A TNConfiguration object.
